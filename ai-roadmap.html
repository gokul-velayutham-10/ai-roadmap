PHASE 01 · MONTHS 1–2
AI Foundations & LLM Literacy
Build the mental models that separate AI engineers from tool users. Understand how LLMs work, not just how to call them. Leverage your existing Python & JS skills immediately.
01
Core Concepts
LLM Fundamentals
  • Transformer architecture (conceptual, not math-heavy)
  • Tokens, embeddings, context windows explained
  • Prompt engineering — zero-shot, few-shot, chain-of-thought
  • Temperature, top-p, sampling — what they mean
  • OpenAI / Anthropic / Gemini APIs — hands-on calls
  • LLM evaluation basics — BLEU, ROUGE, LLM-as-judge
Python Upskill
AI-Focused Python
  • NumPy & Pandas for data wrangling
  • Async Python — critical for streaming LLM apps
  • Pydantic v2 — structured outputs & validation
  • FastAPI — build AI-powered REST endpoints
  • Jupyter notebooks for experimentation
  • HuggingFace Hub — loading + running models locally

PHASE 02 · MONTHS 3–5
LLM Application Engineering
This is where your JS/TS skills become a superpower. RAG, vector databases, and full-stack AI apps are the bread and butter of AI Engineer roles in 2025–26.
02
RAG & Vector Search
Retrieval-Augmented Generation
  • Embedding models — text-embedding-3, Cohere, BGE
  • Vector databases — Pinecone, Weaviate, pgvector
  • Chunking strategies — semantic, hierarchical, sliding window
  • Hybrid search — BM25 + vector fusion
  • Re-ranking with Cohere Rerank or cross-encoders
  • LangChain / LlamaIndex — RAG pipelines
Full-Stack AI
AI App with TypeScript
  • Vercel AI SDK — streaming, tool use, multi-modal
  • Next.js 15 App Router + Server Actions for AI
  • Structured outputs with Zod + OpenAI
  • Streaming UX patterns — SSE, tokens to UI
  • AI SDK RSC — generative UI components
  • Function calling / tool use deep-dive
Orchestration
LangChain & LlamaIndex
  • LCEL (LangChain Expression Language)
  • LlamaIndex query engines and node parsers
  • Memory types — buffer, summary, entity, vector
  • Callbacks & tracing with LangSmith
  • Document loaders — PDF, web, databases
AWS AI Services
Leverage Your AWS Skills
  • Amazon Bedrock — model access & RAG (Knowledge Bases)
  • OpenSearch Serverless as vector store
  • AWS Kendra enterprise search
  • SageMaker Inference endpoints
  • Step Functions for AI workflow orchestration

PHASE 03 · MONTHS 6–9
AI Agents & Automated Pipelines
Agentic AI is the hottest frontier. Companies are racing to automate complex workflows. This is where 6-figure AI roles live in 2025–2026.
03
AI Agents
Agentic Systems
  • ReAct, Plan-and-Execute agent patterns
  • Tool/function calling — design & chaining
  • LangGraph — stateful multi-step agents
  • OpenAI Assistants API + Threads
  • Computer use agents — browser automation with AI
  • Multi-agent systems with CrewAI or AutoGen
Data Pipelines
AI Data Engineering
  • ETL for AI — data ingestion at scale
  • Apache Airflow for AI pipeline scheduling
  • AWS Glue + Bedrock pipelines
  • Structured extraction at scale (Instructor library)
  • Batch processing with OpenAI Batch API
  • Data quality for AI training datasets
Fine-Tuning
Model Customization
  • When to fine-tune vs RAG — decision framework
  • OpenAI fine-tuning API — dataset prep
  • LoRA / QLoRA with HuggingFace PEFT
  • SageMaker training jobs for fine-tuning
  • Evaluation & regression testing post-fine-tune
Observability
LLM Ops Basics
  • LangSmith tracing and dataset management
  • Langfuse open-source LLM observability
  • Cost tracking — token usage dashboards
  • Prompt versioning and A/B testing
  • Guardrails — input/output validation

PHASE 04 · MONTHS 10–12
Production AI Systems & Specialization
Go from building AI features to owning AI systems. Security, scalability, cost optimization, and choosing your specialist track. This is where senior AI Engineer salaries begin.
04
Production
AI in Production
  • Model serving — vLLM, TGI, SageMaker endpoints
  • Caching strategies — semantic cache with Redis
  • Rate limiting & cost control at scale
  • CI/CD for AI — prompt regression tests
  • AI security — prompt injection, data leakage
  • Canary deployments for model updates
Specialist Tracks
